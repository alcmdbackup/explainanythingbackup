# Adhoc Evolution Testing Plan

## Background
Use Playwright to look for usability issues and bugs on the evolution admin dashboard and evolution system generally. This is an ad-hoc exploratory testing project to identify UX problems, broken flows, and edge cases across the evolution admin pages.

## Requirements
- Use Playwright MCP to explore the evolution admin dashboard in headless mode
- Identify usability issues, bugs, and UX problems
- Cover key pages: dashboard, runs, experiments, strategies, arena, prompts, variants, invocations
- Document all findings with screenshots and reproduction steps

## Problem
The evolution admin dashboard has 15+ pages with complex interactions. Manual testing coverage is limited. Need systematic exploratory testing to find usability issues before they impact admin workflows.
